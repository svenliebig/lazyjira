# lazyjira

A minimal CLI for interacting with Jira Cloud in the terminal.

## Installation

### Homebrew (macOS / Linux)

```sh
brew tap svenliebig/tap
brew install lazyjira
```

### Manual

Download the latest release for your platform from the [releases page](https://github.com/svenliebig/lazyjira/releases), extract the archive, and move the binary to a directory in your `PATH`:

```sh
# Example for macOS arm64
curl -L https://github.com/svenliebig/lazyjira/releases/latest/download/lazyjira_<version>_darwin_arm64.tar.gz | tar xz
mv lazyjira /usr/local/bin/
```

Replace `<version>` and the platform/architecture suffix as appropriate.
